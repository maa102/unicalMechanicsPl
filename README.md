# UnicaleMechanics - уникальные механики для сервера Minecraft
Плагин реализует новые механики в майнкрафт, которые мало где можно встретить

## Что добавляет этот плагин:
- Переделку сухого бетона в обычный через котёл, после переделки стака бетона котёл должен опустеть
- При вводе команды открывается менюшка где можно выбрать цвета никнейма
